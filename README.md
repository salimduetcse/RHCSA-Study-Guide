# RHCSA Repository

This repository is dedicated to **RHCSA (Red Hat Certified System Administrator)** preparation. It includes practical Linux administration notes, command references, scripts, lab exercises, troubleshooting examples, and exam-oriented tasks.

The purpose of this repo is to provide a structured hands-on learning path for mastering essential RHCSA topics such as user management, storage, networking, services, security, and shell scripting in Red Hat Enterprise Linux.

## Topics Covered
- Linux Commands
- Users and Groups
- Permissions
- Package Management
- Systemd and Services
- Networking
- Storage and LVM
- SELinux
- Firewall
- SSH
- Cron Jobs
- Bash Scripting
- RHCSA Practice Labs

## Goal
To prepare for the RHCSA exam through consistent hands-on practice and documentation.
